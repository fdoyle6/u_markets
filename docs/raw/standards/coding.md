# Coding Standards

## Overview

This file contains standards for code development in the repository.

## General Standards

1. File should be runnable from any folder (i.e. if a script was run from the base of the repo or the scripts folder, how it runs should not change based on the users cwd at the time that the script was called).
    - There is likely some boilerplate code that will be needed to make this work. At some point I will figure out how to deal with this.
