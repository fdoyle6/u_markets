# Requirements:
- VS Code - VS Code is the expected development environment for this project. It is not required per-se but unless otherwise motivated will be the expectation.
- Windows Operating System - Your computer should run windows. Later changes may expand functionality to include linux, but this is not guaranteed until then.
- Git Bash - Git bash is the expected terminal of use. Command prompt and powershell may be included in the future, but will not be the default build configuration.
